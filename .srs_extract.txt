3. INDUSTRY TEMPLATE ENGINE (VERTICAL AUTO-CONFIGURATION) At sign-up, a new account selects its industry from a preset list. The Platform then automatically provisions the pipeline stages, custom fields, starter template pack, dashboard KPIs, module visibility, and compliance footer text for that industry — turning a generic CRM into a purpose-built one in minutes, not weeks. 3.1 How Industry Selection Works ● Onboarding Wizard asks "What industry are you in?" with the preset list below, or "Start from blank / custom". ● The system auto-provisions: Lead/Deal pipeline stages, custom fields on Leads/Contacts/Deals, a starter Template Pack (Email, SMS, WhatsApp, Document), the Dashboard KPI widget set, and any compliance disclaimer merged into every outbound template footer. ● Industry-only modules are switched on automatically — e.g. a Lenders Panel for Mortgage, Property Listings for Real Estate, Trust Account tracking for Legal — and hidden for industries that don't need them. ● Everything provisioned is a starting point, not a lock-in: an admin can edit any stage, field, template, or KPI afterward from Settings → CRM Configuration. ● Workspace Switcher — a single Nepatronix-built instance can host multiple industry workspaces under one login, so the Connex Group can run Finconnex (mortgage), WorkConnex (recruitment), EduConnex (migration/education), and NatureConnex (trades) as four independently configured, separately branded workspaces on one platform. 3.2 Preset Industry Verticals Industry Pipeline / Stages Key Custom Fields Starter Template Pack Dashboard KPIs Compliance Footer Mortgage & Finance Broking (e.g. Finconnex) New Enquiry → Pre-Qualification → Application Submitted → Conditional Approval → Unconditional Approval → Settled
